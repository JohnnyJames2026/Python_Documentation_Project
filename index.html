Initialization

The first thing to know is what is a variable.

A variable contains data types such as:

  • Integers
  • Strings
  • Booleans
  • Float
  • Lists
  • Tuples

An integer is any number, as long as it is not a decimal.

A string is any character you can type, and it is read with no numeric value.

A boolean are logic data types such as True or False, and (and, or, not)

Floats are basically integers, but they are only decimals. These numbers can go to any decimal place

A list is a group of different data types, and they are all stored within this container. Lists can be parced and contents can be accessed.

A tuple is like a list, but a tuple can not be changed and it is ordered.
