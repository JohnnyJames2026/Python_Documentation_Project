If Statements

The if statement is a conditional control structure that allows you to execute certain code only when a specific condition is met. It evaluates a given expression, and if the result is True, the block of code associated with the if statement runs. If the condition evaluates to False, the program skips the block or evaluates any subsequent elif or else statements.

An example would go as follows:

x = 10

if x > 5:
print("x is greater than 5")

There are also conditions for the if statement called else and elif. If a condition is false, the code will immediately execute the lines following "else". If there is an elif in the conditional chain and the first if was false, then the code will evaluate the elif statement for True or False. If this is wrong, else is the final line ran along with it's following statements.

Continuing from our last example...:

x = 10

if x > 15:
print("x is greater than 5")
elif x == 10:
print("x is equal to 10")
else:
print("x is less than 10")

Loops

Loops allow you to repeat a block of code multiple times, making it easier to handle repetitive tasks without writing the same code over and over. In Python, there are two main types of loops:


  • for loop
  • while loop

The for loop is used to iterate over a sequence (like a list, tuple, dictionary, set, or string) or other iterable objects. With each iteration, the loop assigns the next item in the sequence to a variable and runs the indented block of code.

For example, this is some code that iterates through a list:

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
print(fruit)

#also iterate a certain number of times. This is looping 5 times
for i in range(5):
print(i)

Now, the while loop runs as long as a specified condition remains True. It evaluates the condition before executing the code block, and if the condition is True, it executes the block and then rechecks the condition.

x = 0

while x < 5:
print(x)
x += 1

The only catch to the while loop is you must increment the iterating variable, as the example shows in x. This is the only way to iterate thte loop in steps. Otherwise, this will result in a never ending loop.

Functions

Functions are reusable blocks of code that perform a specific task. They allow you to write code once and use it multiple times, making your programs more organized and modular. In Python, you can define your own functions or use built-in functions.

Defining a Function

You define a function using the def keyword followed by the function name, parentheses, and a colon. Inside the parentheses, you can include parameters that the function can take as input. The function body contains the code that runs when the function is called.

def function_name(parameters):

#function which greets a person
def greet(name):
print("Hello, " + name)

Calling a Function

Once a function is defined, you call it by writing the function name followed by parentheses. If the function takes parameters, you pass the arguments inside the parentheses.

greet("Alice")

Function Parameters

A function can accept any number of parameters, separated by commas. You can also set default values for parameters or use keyword arguments.

def add_numbers(a, b):
return a + b

In this past example, the function returns a value. This means that when the function is called, the value that was calculated will be printed/used.
